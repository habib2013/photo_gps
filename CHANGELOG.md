## 0.0.1

* Initial release of CDL Photo GPS SDK
* Camera integration with high-quality photo capture
* GPS location retrieval with timeout handling
* Address lookup from coordinates (geocoding)
* Location embedding directly on image pixels
* Local storage management
* Comprehensive permission handling
* Offline support (GPS coordinates work without internet)
* Error recovery for timeouts and failures
* Support for Android 5.0+ and iOS 12.0+
